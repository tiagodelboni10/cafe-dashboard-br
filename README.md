# Coffee Market Prices Agent

This project is an agent for fetching and analyzing coffee market prices. It includes data scraping, storage, and basic analysis features.

## Features

- Fetch coffee prices from various sources
- Store data locally
- Perform basic analysis

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the agent: `python main.py`
